Для отображения фрагментов кода используется тег <pre> и <code>:

<!DOCTYPE html>
<html lang="ru">
<head>
<meta charset="UTF-8">
<title>Пример кода</title>
</head>
<body>
<h1>Это заголовок внутри кода</h1>
</body>
</html>
